# LiveChatRoom
server: node.js, typescript, socketIO, express  
client: socket.io-client, bootstrap, jquery

    |-- dist
        |-- client
        |-- server
    |-- src
        |-- client
        |-- server

* Install package
```
$ npm install
```

* Start
```
$ npm run start
```

* Develpoment
```
$ npm run dev
```

Then visit http://127.0.0.1:3000
